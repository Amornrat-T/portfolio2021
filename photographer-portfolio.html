Brief

He wanted this website to be an online photo gallery website. What he wants is to have a website that is easy to use and informative, complete and stylish. This website is created using wix.com to provide the functionality of the customer. In the event that the customer wants to add or remove content on the website.

Website owner

Dustin Smith. He is American who live and work in Thailand. He work in data analytics but find joy going out in taking photos.

Client Research

Send questionnaires to customers to analyze customer needs and request a reference website from customers to be adapted to the website design.

Site map and wireframe

Bring customer brief to design your website and other elements. And plan the format of the website and create a sitemap and wireframe to offer customers to bring feedback to improve.

Prototype

Create a website prototype on the XD program so that customers can try it out and adjust it to the customer’s needs before building it.

Site map

Create layouts and pages on the website to cover all content. There are 3 main pages and 4 secondary pages where users can access various galleries from the home page.

Style guide

The main color of this website is black and white. In order to get the most attention and focus on the image on the website, the layout is simple, except for the first page with a new layout. To make users more eye-catching and easy to remember the website by placing images in a zigzag pattern to guide the user to scroll down to reach the last part of the page.

Problem

After the website is up and running, we periodically monitor the access and usage of our website visitors and discover gallery access issues from the homepage. When the user scrolls down to see the image on the homepage and tries to access the other images in that category but is unable to access it and leaves the website faster than expected.

Solving

I was aware of all the information and analyzed the problem that the button to access another gallery was not clear enough. So I have to increase access. When users want to view images in that category, they can click on it immediately. After the problem is solved, the traffic of the site is increasing and the amount of time it takes for the user to visit the site is longer.

Give advice to customers

Bring experience and knowledge to guide customers in designing and formatting the website for the best results.

Monitor the work of the website

After the website has been activated. The work of the website must be tracked periodically in order to study the behavior of visitors to the website and improve it.

Analysis and problem solving

Due to the tracking of user behavior of website visitors, it will discover problems in using. So I had to analyze the problem and fix it promptly, and I need to follow up after it has been resolved.
